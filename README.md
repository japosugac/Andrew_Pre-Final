## Project Overview

This project is a simple, beginner-friendly website that introduces students to **Generative AI (GenAI)**.
It contains four sections that explain GenAI concepts, ethical guidelines, and a real-world analogy.

The website is created for **CSDC100 Pre-Finals** and is designed for students who are new to AI and its applications.

It covers:

- Frequently Asked Questions (FAQs) about Generative AI
- Ethical use of AI in academic, research, and daily life
- An analogy on the use of AI in creative and development work
- A reflection on integrity, skill, and responsible use of AI tools
